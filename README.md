# TP2 Maxime Bourret 

### Description

Application par défaut Vite et React.
Servie dans un conteneur Docker avec Node.js ([serve](https://www.npmjs.com/package/serve)).

### Commande pour construire l’image 

```bash
docker build -t image-react .
```

### Commande pour lancer le conteneur 

```bash
docker run -p 3000:3000 image-react
```


Lien dockerhub : https://hub.docker.com/repository/docker/maximebourret/image-react/general
